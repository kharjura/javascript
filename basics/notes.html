DOM functions: document.getElementById("idName"):select an element based on an id document.getElementsByclassName("className"):select multiple elements based on className The elements are stored in the form of an array [list] document.getElementsByTagName("tagName"):select elements based on tag name given document.querySelectors("css-selectors"):selects an element based on the given css query ------------------------------------------------------------ function to create elements: document.createElement("element-Name"):creates a new element in the dom document.appendchild("element-Name"):adds the newly created element to the Dom tree
